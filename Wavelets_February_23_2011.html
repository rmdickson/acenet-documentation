Wavelets February 23, 2011

CCDB User Registration Reminder

Project Leaders should have received an email with instructions for having their group members register with the CCDB. We would like to have all users registered by March 1, 2011. After this date, any account not registered with the CCDB may be unable to access ACENET resources. Individual users can register here.

New Purchase Survey

ACENET has some funds from its National Platform Fund award to make an addition to the Glooscap cluster at Dalhousie. We've created a very short survey so our users can help direct this purchase. The survey will close February 28. Please see here.

Planned Maintenance

Placentia has a planned maintenance scheduled for February 28 until March 4. Grid Engine will not schedule any job whose run time (h_rt) extends into the outage period. Latest information on this website.

Software Update

  • VASP has been updated to version 5.2.11. The older version is still available.
  • GAMBIT has been discontinued by Ansys. Our existing license will expire on March 6, 2011. Gambit has been replaced by similar functionality in the new versions of Fluent/CFX. Anyone needing assistance with this transition, please email support.
