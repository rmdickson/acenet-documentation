MATLAB Distributed Computing Server

ACENET's license for the MATLAB Distributed Computing Server (DCS) expired on 2018 June 30 and will not be renewed. The MATLAB DCS is no longer supported on ACENET equipment.

If you wish to continue using MATLAB, your options are:

  1. Migrate to MATLAB on national systems.
  2. Use the MATLAB Runtime and compiler.
