Vim

Description
Vim is a text editor that is upwards compatible to vi. It can be used to edit all kinds of plain text. It is especially useful for editing programs.
Modulefile
vim
Documentation
If you are completely new to vim, we recommend running vimtutor which provides an automated tutorial.
Vim also has an excellent built-in help facility --- just type :help inside vim.

Tips
  • :help is your friend
  • :w saves the changes you've made
  • :q! gets you out without saving any changes you have made
  • If you don't like the text coloring that happens for some source code files, :syn off will turn it off.
Usage
$ vimtutor
$ vim
$ vi
