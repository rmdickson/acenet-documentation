RStudio

The easiest way to run RStudio on Siku or Argo is to start RStudio via JupyterHub. JupyterHub is a service which allows you to use the compute nodes of a cluster via your web browser.

Open a browser tab on one of these addresses, depending on the cluster you're using:

  • https://siku-portal.alliancecan.ca/jupyter
  • https://argo-portal.alliancecan.ca/jupyter

At the Server Options page, choose "JupyterLab" for the user interface. Then follow the guidance here.

You must have an Alliance account in order to use JupyterHub. We currently cannot support JupyterHub for local Siku accounts.
