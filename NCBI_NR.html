NCBI NR

The NCBI NR database is provided on ACENET for RAPSearch and BLAST. The database was updated last on 2017-02-23.

The NR database directory contains the following files

  • nr - FASTA format database downloaded from here
  • BLAST-formatted files nr.XX.pXX (protein sequences) downloaded from here
  • nr_raps2 - RAPSearch-reformatted database
